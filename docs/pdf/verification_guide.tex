\section{Introduction}
This document describes a complete verification environment for a dual-channel adder design using
SystemVerilog, Cocotb, and Verilator. It includes architecture, coverage, regression methodology,
and waveform generation.

\section{Coverage Model}
Coverage includes operand bins, pattern bins, overflow bins, illegal bins, temporal bins, and
cross coverage. Threshold gating requires 75\% minimum coverage.

\section{Regression}
Regression is executed using:
\begin{verbatim}
./tools/run_regression.sh
\end{verbatim}

\section{Waveform Generation}
Waveforms are enabled using:
\begin{verbatim}
export VERILATOR_TRACE=1
\end{verbatim}

Waveforms appear in:
\begin{verbatim}
sim_build/dump.vcd
\end{verbatim}
