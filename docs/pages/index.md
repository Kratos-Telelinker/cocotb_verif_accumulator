# Dual-Channel Adder Verification Environment

Welcome to the documentation site for the **Dual-Channel Adder Verification Environment**, a
production-grade verification framework built using **SystemVerilog**, **Cocotb**, and **Verilator**.

This project demonstrates modern verification methodology including:

- Multi-agent stimulus (Channel A & B)
- Directed, constrained-random, and randomized testing
- Functional coverage (coverpoints, crosses, temporal bins)
- Illegal condition detection
- Scoreboard-based checking
- Regression automation with coverage gating
- Verilator waveform dumps (VCD/FST)
- Full documentation package (PDF + GitHub Pages)

Use the navigation sidebar to explore the full documentation.
